Python Sul 2017

Data: 08-09-2017 Categoria: Eventos Tags: eventos python

1° Conferência das communidades Python da Região Sul

A Python Sul reuniu pessoas durante 3 dias, 8,9 e 10 de setembro, para discutir sobre software livre, práticas de desenvolvimento e Python.

Organização

A Python Sul 2017 é organizada pela comunidade Python do Rio Grande do Sul, com a ajuda da comunidade de Santa Catarina e Paraná.
