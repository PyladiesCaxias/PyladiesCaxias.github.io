Django Girls Caxias do Sul

Date 07-09-2017 By Pyladies Category Eventos. Tags eventos django_girls python pyladies poa

Django Girls é uma oficina de programação gratuita para mulheres. Caxias do Sul teve sua 1° edição no dia 07 de setembro de 2017, antecedendo a Python Sul 2017. Neste 1° Django Girls Caxias do Sul as mulheres conseguiram conhecer um pouco de programação seguindo o tutorial do Django Girls.

Site produzido em Pelican , com a tecnologia Python .
